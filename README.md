	2.	Привязка 2D плана к 3D сцене
	•	Преобразуйте 2D-координаты в 3D (например Z=0 уровень пола, стены — экструзия).
	•	Создайте базовую геометрию помещения (стены, пол) либо вручную, либо с помощью генератора.

	3.	Размещение объектов (мебели, лотов)
	•	Определите правила: какие объекты могут стоять в какой зоне, ориентиры, расстояния от стен и т.п.
	•	Используйте scene_synthesizer либо собственный модуль “rules_stub” для размещения объектов.
	•	При необходимости воспользуйтесь библиотекой CadQuery для генерации специфичных моделей.

	4.	Экспорт сцены / моделей
	•	После генерации сцены экспортируйте её в JSON-формат или .glb/.usdz/.fbx, как это описано в архитектуре проекта.
	•	Убедитесь, что соблюдены ваши контракты: SceneGraph, Zone, Lot, и тесты проверяют, что, например, ≥10 объектов размещено.
    
	5.	Интеграция в pipeline
	•	Ваш “layout_stub” может принимать SVG параметр, генерировать сетку зон.
	•	“rules_stub” — размещает объекты.
	•	“export_stub” — сохраняет scene_mock.json.
	•	Тесты: проверка наличия как минимум одной зоны, множества объектов и корректного формата.
